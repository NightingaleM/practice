    • 题目1： ajax 是什么？有什么作用？
答：
AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。
Ajax的主要优势就是对整个页面的请求以异步的方式发送到服务器。而服务器不会用整个页面来响应请求，
它会在后台处理请求，并且只更新页面中的一小部分。与此同时，用户还能继续浏览页面并与页面交互。
				
    题目2： 前后端开发联调需要注意哪些事情？后端接口完成前如何 mock 数据？
答:
	1.前后端开发联调需要注意以下事情： 
		约定数据：约定好页面需要的数据和数据类型
		约定接口：约定接口的名称、请求的参数、相应的格式（成功或失败分别返回什么）
		将约定好的内容整理成接口文档和规范
	2.后端接口完成前如何 mock 数据？ 
		可以使用server-mock或mock.js搭建模拟服务器，进行模拟测试
		mock.js的github仓库
		server-mock的github仓库
		使用XAMPP等工具，编写PHP文件来进行测试
